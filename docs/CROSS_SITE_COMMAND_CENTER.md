## Cross-Site Daily Routine (10–20 min)

For each site:
- [ ] Homepage returns 200
- [ ] `/robots.txt` returns 200
- [ ] `/sitemap.xml` returns 200
- [ ] Search Console has no critical new issues
- [ ] New pages submitted for indexing

---

## Cross-Site Weekly Routine (45–60 min)

For each site:
- [ ] Review Search Console queries (positions 6–20)
- [ ] Improve titles/meta for pages with impressions but weak CTR
- [ ] Add internal links from homepage/hub pages to new guides
- [ ] Publish or refresh at least 1 content page
- [ ] Confirm sitemap includes all new pages

---

## Shared SEO Standards

Apply to both websites:
- Canonical host consistency (single preferred domain)
- Unique metadata per indexable page
- One H1 per page
- Internal links between hubs and guides
- JSON-LD (Organization + WebSite + page-specific where relevant)
- Search Console verification + sitemap submission

---

## Cross-Site KPI Targets (90 days)

- 10+ long-tail keywords in top 10 per site
- 3–5 high-intent keywords in top 5 per site
- Week-over-week growth in non-brand impressions/clicks
- Measurable improvement in CTR for optimized pages
