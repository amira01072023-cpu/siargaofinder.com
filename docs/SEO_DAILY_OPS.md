# SEO Daily Ops — Siargao Finder

Use this checklist to keep rankings improving consistently after launch.

## Daily (10–15 min)

- [ ] Confirm site health
  - `https://www.siargaofinder.com/` loads
  - `https://www.siargaofinder.com/robots.txt` loads
  - `https://www.siargaofinder.com/sitemap.xml` loads

- [ ] Check Search Console
  - Open Performance report (last 24h / 7d)
  - Note major drop/spike in clicks or impressions
  - Confirm no new critical coverage errors

- [ ] Submit newly published URLs
  - Use URL Inspection for any new page
  - Request indexing for priority pages

## Weekly (30–45 min)

- [ ] Performance review
  - Identify queries in positions **6–20**
  - Improve title/meta for those pages (CTR + relevance)

- [ ] Content iteration
  - Refresh 2 pages with improved intros/FAQs/internal links
  - Add at least 1 new guide or city/category enhancement

- [ ] Internal linking pass
  - Ensure new pages are linked from homepage/category/city hubs
  - Add contextual links between related guides and listings

- [ ] Technical checks
  - Validate sitemap contains newly added pages
  - Confirm canonical URLs are `https://www.siargaofinder.com`
  - Spot-check structured data presence on homepage/listing pages

## Monthly (60–90 min)

- [ ] Ranking trend review for target keywords
- [ ] Backlink progress check (new referring domains)
- [ ] Update top-performing pages with fresher local context (month/year)
- [ ] Remove/merge thin pages if needed

## Priority Targets (first 90 days)

- Top 10 for 10+ long-tail keywords
- Top 5 for 3–5 high-intent terms
- 20+ quality referring domains
- Consistent week-over-week growth in non-brand impressions/clicks
